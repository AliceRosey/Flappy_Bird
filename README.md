# Flippy Bird - Python Arcade Game 

Flippy Bird is a 2D arcade-style game developed using Python and Pygame.  
The player controls a bird that must fly through moving pipe obstacles while avoiding collisions and staying airborne.

The game includes animated bird movement, gravity physics, scoring mechanics, collision detection, and 4 progressive difficulty levels that increase the challenge as the player advances.

# Features

- Animated bird character
- Smooth jump and gravity mechanics
- Dynamic moving pipe obstacles
- Collision detection system
- Real-time score tracking
- 4 progressive game levels
- Increasing difficulty and pipe speed
- Game over and restart system
- Scrolling environment animation
- Keyboard-based controls

# Game Levels

|------|------|------|------|
| 1 | Beginner | Slow | Large |

| 3 | Medium | Fast | Smaller |
| 4 | Hard | Very Fast | Small |

# Controls

| Key | Action |
|------|------|
| SPACE | Jump / Start Game |
| R | Restart Game |
| M | Back to menu |
