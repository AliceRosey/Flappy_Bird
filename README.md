# Flippy Bird - Python Arcade Game

Flippy Bird is a 2D arcade-style game developed using **Python** and **Pygame**.
The player controls a bird that must fly through moving pipe obstacles while avoiding collisions and staying airborne.

The game includes animated bird movement, gravity-based jumping, scoring mechanics, collision detection, and 4 progressive difficulty levels that increase the challenge as the player advances.

## Features

* Animated bird character
* Smooth jump and gravity mechanics
* Dynamic moving pipe obstacles
* Collision detection system
* Real-time score tracking
* 4 progressive game levels
* Increasing difficulty and pipe speed
* Game over and restart system
* Scrolling environment animation
* Keyboard-based controls

## Game Levels

| Level | Difficulty | Pipe Speed | Gap Size |
| ----- | ---------- | ---------- | -------- |
| 1     | Beginner   | Slow       | Large    |
| 2     | Easy       | Medium     | Medium   |
| 3     | Medium     | Fast       | Smaller  |
| 4     | Hard       | Very Fast  | Small    |

## Controls

| Key   | Action            |
| ----- | ----------------- |
| SPACE | Jump / Start Game |
| R     | Restart Game      |
| M     | Back to Menu      |

## Installation

Install the required dependency:

```bash
pip install pygame
```

Run the game:

```bash
python main.py
```

> If your main file has a different name, replace `main.py` with your actual Python file name.
